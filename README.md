# aland-slides-generator (V0.1)

一个面向 0 代码基础用户的网页幻灯片生成工具。  
你只要填写表单，就能生成一个**单文件 HTML 幻灯片**，支持 16:9、左右键翻页、页码和简单动画。

---

## 1. 功能清单（V0.1）

- Streamlit 网页界面
- 页面标题：**艾兰得网页幻灯片生成器**
- 支持输入：
  - 汇报标题
  - 汇报副标题
  - 目标受众
  - 幻灯片大纲（每行一页）
- 支持 3 种风格：
  - 商务科技风
  - 艾兰得蓝色风
  - 浅绿色健康风
- 点击“生成幻灯片”后输出单个 HTML 文件
- HTML 幻灯片支持：
  - 16:9 比例
  - 每页一屏
  - 左右键翻页
  - 底部页码
  - 简单转场动画
- 页面内 HTML 预览
- 下载 HTML 按钮

---

## 2. 运行环境

- Python 3.10 或 3.11（推荐）
- Windows / macOS / Linux 均可

---

## 3. 本地运行（0 基础版）

### 第一步：下载项目
如果你使用 Git：
```bash
git clone <你的仓库地址>
cd aland-slides-generator
```

如果你是直接拿到项目压缩包：
- 解压后进入 `aland-slides-generator` 文件夹。

### 第二步：安装依赖
```bash
pip install -r requirements.txt
```

### 第三步：启动网页
```bash
streamlit run app.py
```

启动后，终端会显示一个本地地址（通常是 `http://localhost:8501`），浏览器打开即可使用。

---

## 4. 使用方法

1. 填写“汇报标题 / 汇报副标题 / 目标受众”。
2. 在“大纲”中每行写一个页面主题（每行生成一页）。
3. 选择风格。
4. 点击 **生成幻灯片**。
5. 页面会显示预览，并可点击 **下载 HTML 文件**。

下载后的 `.html` 文件可以直接双击打开播放。

---

## 5. 部署到 Streamlit Cloud

### 5.1 准备代码仓库
1. 把本项目推送到 GitHub 仓库。
2. 确保仓库根目录包含：
   - `app.py`
   - `requirements.txt`

### 5.2 在 Streamlit Cloud 部署
1. 打开：https://share.streamlit.io/
2. 使用 GitHub 账号登录。
3. 点击 **New app**。
4. 选择你的仓库和分支。
5. Main file path 填：`app.py`
6. 点击 **Deploy**。

几分钟后会得到一个公网访问链接，你可以发给团队直接使用。

---

## 6. 常见问题

### Q1：点击生成没反应？
- 先确认“大纲”至少有一行非空内容。

### Q2：打开 HTML 后不能翻页？
- 请点击页面任意区域，让页面获得焦点后再按左右键。

### Q3：中文显示异常？
- 建议使用新版 Chrome 或 Edge 浏览器打开 HTML。
